# 🤖 Sirius - Sistema Inteligente de Respostas Integradas e Unificadas

O **Sirius** é um assistente virtual inteligente e zoeiro, desenvolvido em Python. Ele utiliza IA para processamento de linguagem natural e síntese de voz de alta fidelidade.

## 🚀 Tecnologias Utilizadas
* **Google Gemini API**: O cérebro do sistema.
* **ElevenLabs API**: Voz de alta qualidade (Liam).
* **LangChain**: Gerenciamento de memória e histórico de conversas.
* **PySide6**: Interface gráfica (Esfera 3D).
* **SpeechRecognition & PyAudio**: Para captar e entender a voz do usuário.

## 🛠️ Como rodar o projeto
1. Clone o repositório.
2. Crie um arquivo `.env` dentro da pasta `config` seguindo o modelo do `.env.example`.
3. Instale as dependências: `pip install -r requirements.txt`.
4. Execute o arquivo: `python src/main_voz.py`.
